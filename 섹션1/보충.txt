forEach, map, filter, reduce
모두 고차 함수(매개변수로 함수를 받음)
배열을 돌면서 각 요소마다 콜백함수를 호출

a = [10, 11, 12, 13, 14, 15];

forEach(함수, 콜백함수에서 this로 사용할 것)
a.forEach(function(v,i){
    console.log(v,i)
})
=> v에는 배열의 요소들이, i에는 for문의 i가 찍힘.
=> 10, 0 / 11, 1 / 12, 2 / 13, 3 / 14, 4 / 15, 5
* function(v,i){
    console.log(v,i)
}
=> 이 부분이 콜백함수

map은 배열을 이용해서 새로운 배열 생성 및 반환.
매개변수로 받은 콜백함수가 리턴한 값을 요소로 하는 새로운 배열 반환.
map이 생성한 배열은 원본 배열과 길이가 같으며 비어 있는 인덱스에는 undefined가 찍힌다.

filter는 원본 배열에서 콜백함수 연산 결과가 true인 요소만 반환.

reduce(콜백함수(누적값, 배열 요소), 초기값)
콜백함수에서 리턴한 값이 다시 콜백함수의 첫 번째 인자로 들어간다.
reduce는 배열을 생성하는 것이 아니라 값을 생성해서 반환한다.
